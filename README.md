# SQL query checker

## Usage

Check single sql file:

```shell
sqlcheck.py <file> <answer>
```

Check all sql files under the specified directory:

```shell
sqlcheck.py <directory> <answer>
```

## Other Utilities

### Dump

```shell
sqldump.py <file | directory>
```

### Format

```shell
sqlformat.py <file | directory>
```
